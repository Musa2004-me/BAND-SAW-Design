Band saws represent essential equipment in modern manufacturing and workshop 
environments, providing versatile cutting capabilities for various materials ranging from soft 
woods to hardened metals. This project focuses on designing a mechanical band saw that 
balances performance, safety, and cost-effectiveness while incorporating advanced 
manufacturing technologies. 


The fundamental design parameters established for this project include: 


 Main Motor Power: 15 kW three-phase AC motor 



 Saw Wheel Diameter: 1250 mm 


 Operating Speed Range: 600-900 RPM (variable through inverter control) 


 Maximum Saw Band Length: 8500 mm 


 Saw Band Specifications: 150 mm width, 1.25 mm thickness
